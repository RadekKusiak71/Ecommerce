# Ecommerce
Ecommerce site project
##Technologies used:
  - Django
  - Django Rest Framework
  - Html
  - Css
  - Bootstrap
##Instalation:
  1. Clone repository from git
  2. Create virtualenv
    ```
    virtualenv venv
    ```
  3. Activate virtualenv and run command:
     ```
     pip install -r requirements.txt
     ```
